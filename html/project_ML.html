Cardiovascular Disease Classification

Purpose:

This is a project that aim to read ECG data as input and classifies different cardiovascular disease using deep neural network.
Machine learning helps doctors to classifies different diseaeses automatically and robustnessly.
This reduces human errors and help patients 24/7.

Pre-processing

Signals are truncated to 10s and filtered with 3rd order butterworth low pass filter of cutoff frequency at 10hz.

Model:

The final model is a deep residual neural network with LSTM attention mechanism. The input shape is 12x5000 for 12 leads ECG signals and 5000
